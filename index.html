Bienvenido/a

Este sitio web está pensado para ayudarte a crear sitios accesibles para personas con discapacidad visual, auditiva y motora/kinestésica.

¿Qué es la accesibilidad web?

La accesibilidad web busca que todas las personas, sin importar sus capacidades, puedan navegar, comprender y usar un sitio web de manera efectiva. Esto incluye a usuarios con discapacidades visuales, auditivas o motoras.

Perceptible

La información debe presentarse de manera clara. Esto incluye buen contraste, texto alternativo y contenido visible para todos.

Operable

La página debe poder navegarse con teclado, sin movimientos precisos y sin obstáculos.

Comprensible

El contenido debe ser fácil de entender: lenguaje claro, consistente y sin confusiones.

Errores comunes de accesibilidad

Estos son algunos de los errores más frecuentes al diseñar sitios web. Todos tienen un impacto negativo en la experiencia de usuarios con discapacidad visual, auditiva o motora.

❌ Bajo contraste

El texto difícil de leer afecta principalmente a personas con baja visión o sensibilidad al contraste. Usa una relación de al menos 4.5:1.

❌ Imágenes sin texto alternativo

Las personas que usan lectores de pantalla no pueden comprender el contenido visual si no se proporciona una descripción significativa.

❌ No se puede navegar con teclado

Muchos usuarios dependen exclusivamente del teclado para navegar. Asegúrate de que todos los elementos sean alcanzables con Tab.

❌ Elementos sin etiqueta o nombre accesible

Botones o enlaces sin etiquetas claras confunden a los lectores de pantalla. Usa textos descriptivos y evita "clic aquí".

¿Por qué es importante la accesibilidad?

La accesibilidad web permite que todas las personas puedan usar un sitio, independientemente de sus capacidades, dispositivos o contexto. También ayuda a cumplir estándares como las WCAG y mejora la experiencia para todos los usuarios.

Cómo usar este sitio

Este portal está diseñado para ayudarte a crear experiencias digitales inclusivas de manera clara y guiada. Puedes explorar su contenido según tus necesidades o nivel de conocimiento. Las secciones principales son:

  • Guías por discapacidad: Recomendaciones prácticas, ejemplos reales y errores frecuentes organizados por tipo de discapacidad (visual, auditiva y motora/kinestésica). Cada guía explica cómo adaptar tu sitio para que todos puedan navegarlo sin barreras.
  • Videos de ejemplo: Clips accesibles con audiodescripción, subtítulos y closed captions para ayudarte a visualizar cómo se implementa la accesibilidad en situaciones reales.
  • Recursos y herramientas oficiales: Enlaces confiables como WCAG, WebAIM y WAVE que te permiten validar, mejorar y profundizar en los estándares internacionales de accesibilidad.
  • Errores comunes que debes evitar: Aprende qué prácticas afectan la accesibilidad y cómo corregirlas de manera sencilla y efectiva.
